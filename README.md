# Millo Caffè

A premium, luxury-inspired coffee shop website concept built with semantic HTML5, modern CSS3, and vanilla JavaScript.

## Structure

- index.html — main landing page
- assets/css/variables.css — design tokens and theme variables
- assets/css/style.css — core visual system and layout
- assets/css/responsive.css — mobile and tablet styles
- assets/js/app.js — navigation and UI interactions
- assets/img/ — image assets
- assets/icons/ — SVG or icon assets
- components/ — reusable components for future expansion

## Design System

- Colors: #111111, #F7F7F5, #C5A46D, #4B3427
- Typography: Playfair Display for headings and Poppins for body copy
- Style: elegant, warm, premium coffeehouse aesthetic

## Future PHP Integration

The structure is intentionally semantic and organized so sections and content blocks can be replaced or included via PHP partials later.

## Run locally

Open index.html in your browser or serve the project from a local web server such as XAMPP.
