# Millo Caffè

Site institucional premium para uma cafeteria italiana com identidade sofisticada, inspirada em marcas como Illy, Lavazza, Starbucks Reserve e Ralph's Coffee.

## Estrutura do projeto

- index.php
- quem-somos.php
- cardapio.php
- contato.php
- assets/css/base.css
- assets/css/layout.css
- assets/css/components.css
- assets/css/pages.css
- assets/css/responsive.css
- assets/js/app.js
- assets/js/menu.js
- assets/js/slider.js
- assets/js/reveal.js
- assets/images/
- assets/icons/
- components/

## Características

- HTML5 semântico
- CSS3 com variáveis e utilidades reutilizáveis
- JavaScript vanilla para interações e animações
- Layout responsivo com CSS Grid e Flexbox
- Navegação premium, hero full screen e animações com Intersection Observer
- Preparado para futura integração com PHP
